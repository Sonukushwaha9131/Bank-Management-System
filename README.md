# 🏦 Bank Management System

A modern Bank Management System developed using Python and Streamlit. This project allows users to create bank accounts, securely log in, deposit and withdraw money, view transaction history, update profile details, and manage accounts through an interactive web interface.

---

## 🚀 Features

### 👤 Account Management
- Create New Bank Account
- Secure Login using Account Number & PIN
- Update Profile Information
- Delete Account

### 💰 Banking Operations
- Deposit Money
- Withdraw Money
- Balance Inquiry
- Account Dashboard

### 📊 Transaction Management
- Store Transactions in JSON Database
- View Transaction History
- Track Deposits and Withdrawals

### 🔒 Security Features
- Unique Account Number Generation
- PIN-Based Authentication
- Session Management

---

## ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/your-username/Bank-Management-System.git
```

### Move into Project Folder

```bash
cd Bank-Management-System
```

### Install Dependencies

```bash
pip install streamlit
```

### Run Project

```bash
streamlit run app.py
```
